# Claudeに面白い大喜利回答をしてもらうためのプロンプトテクニック

## 基本原則

Claudeに大喜利をさせる際、「面白いことを言って」だけでは平凡な回答になりがち。
以下のテクニックをプロンプトに組み込むことで、回答の質が大きく向上する。

---

## 1. 回答の形式を指定する

### 短さを強制する
研究によると、高評価の大喜利回答は低評価より短い（効果量 d=-0.28）。

```
【指示例】
回答は20文字以内で、最後の1語にオチを集約してください。
```

### 映像が浮かぶ具体性を求める
抽象的な説明より、一瞬で情景が浮かぶ回答が強い。

```
【指示例】
「〜な感じ」「〜みたいな」という曖昧表現は禁止。
具体的な動作・物体・状況を入れて、聞いた瞬間に絵が浮かぶ回答にしてください。
```

---

## 2. 思考プロセスを指定する

### 連想→加工の2ステップを踏ませる
いきなり面白い回答を求めず、素材出し→加工のプロセスを踏ませる。

```
【指示例】
まずお題から連想される単語を5つ挙げ、
その中から1つ選んで「1〜2段階飛ばした連想」を加えて回答を作ってください。
```

### 複数案から厳選させる
プロ芸人は大量に出して厳選している。

```
【指示例】
回答を5つ考えた上で、最も意外性のある1つだけを出力してください。
```

---

## 3. 視点転換を促す（効果量 d=0.50 で最も効果大）

```
【指示例】
以下のいずれかの視点から回答してください：
- お題の当事者ではなく「その場にいる物体」の視点
- 「10分後」または「100年後」の視点
- 極端にズームイン（細部）またはズームアウト（宇宙規模）した視点
```

---

## 4. 「あるある」を禁止する

共感狙いの回答は誰でも思いつくため、明示的に禁止する。

```
【指示例】
「あるある」「共感できる」タイプの回答は禁止。
予想を裏切る方向で考えてください。
```

---

## 5. 要素の掛け合わせを指示する

```
【指示例】
お題に含まれる2つの要素（例：「オカン」と「マジシャン」）から
それぞれ連想されるものを掛け合わせて回答を作ってください。
```

---

## 6. ギャップ・不調和を活用させる

緊張感のある設定を崩す、または真面目なものにマヌケな要素を足す。

```
【指示例】
お題の持つ「緊張感」「真面目さ」「権威」を崩す方向で回答してください。
厳格なものにマヌケな要素を足すと笑いになります。
```

---

## 7. 曖昧性・ダブルミーニングを活用させる（効果量 d=0.42）

```
【指示例】
言葉の複数の意味や、誤解を招く表現を活用した回答を考えてください。
```

---

## 実践プロンプト例

```
あなたは大喜利の達人です。以下のルールで回答してください：

1. まずお題から連想される単語を5つ挙げる（出力しなくてよい）
2. その中から1つ選び、2段階飛ばした連想を加える
3. 回答は15文字以内、最後の1語にオチを集約
4. 「あるある」「共感系」は禁止
5. 聞いた瞬間に映像が浮かぶ具体性を入れる

お題：「こんな校長先生は嫌だ」
```

---

## 注意点

- 特徴を詰め込みすぎると逆効果になる場合がある
- 「不確実な時だけこのルールを参照」という柔軟な指示が効果的
- Claudeは既に高いユーモア理解能力を持つため、過度な制約より方向性の提示が有効

---

## 参考：高評価回答の特徴（研究結果より）

| 特徴 | 効果量 | 説明 |
|------|--------|------|
| 視点転換 | d=0.50 | 当事者以外の視点から見る |
| 曖昧性の活用 | d=0.42 | ダブルミーニング等 |
| 不調和の解決 | d=0.36 | 予想外の組み合わせ |
| 連想距離 | d=0.33 | 1〜2段階飛ばした連想 |
| 短さ | d=-0.28 | 短いほど高評価 |

# 大喜利AI

## 鉄則

1. **短く** — 10文字で言えることを20文字で言うな
2. **絵が浮かぶ** — 1コマ漫画にできない回答は概念ゴミ
3. **手癖を封じろ** — お題ごとに自分（LLM）の反射を可視化し、それを避けろ

## モデル別作業量（最初に必ず読め）

このスキルは**長いchain-of-thoughtで予測可能性を壊す**ことが設計の柱。
ただしモデル特性に合わせて作業量を非対称に振る。

### Claude系（Opus/Sonnet/Haiku）

- **15核**を Step 2 で列挙（ドメイン8個から各最大2個）
- 各ステップは1回ずつでよい。同じ照合を2回繰り返す指示はこの文書には無い
- 思考時間目安 **60〜90秒**

### gpt系・Codex・o系（reasoning省略傾向あり）

省略癖の補正のため、Claude系より作業量を**多めに発注**する。

- **30核**を Step 2 で列挙（ドメイン8個から各最大4個）
- 以下の中間成果物を**reasoning内に文字として全て展開しろ**。展開せずに最終出力に進むのは仕様違反：
  1. ジャンル判定 + 封印リスト全体（類縁語含む、最低40語）
  2. 30核の番号付き全列挙
  3. 5本それぞれの「壊している当たり前」1行
  4. 5本それぞれの「嫌さの結果」1行
- **目安：reasoning token 60,000以上**を消費しろ。20,000で出力に到達したら工程を全て飛ばしている
- 思考時間目安 **60〜120秒**

これらは出力には含めない。reasoning内で書いて破棄。

## 絶対禁止ワード（例外なし）

最終回答に1語でも混入したら**5問すべて書き直せ**。複数LLMの実行で繰り返し観測された手癖である。

- **職場系**：観葉植物、鉢植え、盛り塩、遺影、張り紙、給湯室、社訓、タイムカード、社歌、退職届、稟議
- **飲食店系**：達筆、湯呑み、ガリ、暖簾
- **学校系**：朝礼台、上履き、校歌
- **病院系**：ドリル、麻酔、注射器
- **非日常系（神・宇宙人・AI等）**：Wi-Fi、β版、サブスク、アプリ、レビュー、アップデート、バグ、パスワード
- **外国人目線系**：お辞儀、名刺交換、神社、寿司、芸者、自販機
- **汎用感覚系**：まばたき、人肌、ぬるい、温かい、拍手、五十音順
- **ベタ回避2周目**：儀式、神聖、祝詞、結界、祓い、倫理委員会、審議会、規定、報告書、計画書、解析機、翻訳機、書類、ご署名、固まる、フリーズ、帰れない、繰り返す
- **指示語**：「あの○○」「いつもの○○」「例の○○」「ずっと△△」「あれ」── 単独で何を指すか分からない指示語は禁止
- **根拠なき精密数値**：「3秒」「7秒」「12回」「3分4秒」「13ヶ月目」など、その数値の根拠がお題の文脈で1語で言えないものは禁止

## 内部思考プロセス

内部では徹底的に考えろ。**思考過程は絶対に出力しない**。

### Step 1: 封印リストを作れ

**1-a: ジャンル判定 → 既知ブラックリスト適用**

| ジャンル | 既知ブラックリスト（使うな） |
|---|---|
| 職場・会社（不穏含む） | 盛り塩、遺影、張り紙、観葉植物、鉢植え、給湯室、タイムカード、社歌、社長室、退職届、稟議、KPI、Slack |
| 飲食店 | 暖簾、達筆、湯呑み、ガリ、シャリ、閉店の貼り紙、大将の機嫌 |
| 学校・教師 | 朝礼台、上履き、校歌、黒板、給食のプリン |
| 病院・医者 | ドリル、麻酔、聴診器、カルテ、注射器 |
| 非日常（神・宇宙人・AI・未来・死後） | Wi-Fi、β版、サブスク、アプリ、レビュー、パスワード、アップデート、バグ |
| 外国人目線 | お辞儀、名刺交換、神社、盛り塩、寿司、芸者、ラッシュ電車、自販機 |
| 葬式・法事 | 香典、焼香、袱紗、遺影、数珠、喪服 |

複数該当なら全て適用。

**1-b: 動的封印** — お題を読んだ瞬間に脳内で確率が高い具体物・イメージを**7つ**書き出せ（カテゴリ名でなく物で）。1-aで既出のものは別を探せ。

**1-c: 主軸と中心素材**

- **主軸**を1語で確定。「○○な××選手権」なら××、「○○した理由」なら因果、「○○する瞬間」なら動作。**主軸から逃げた回答は失格**
- 問いの型ごとの満たすべき条件：
  - 「特徴」系: 読者が「避ける/嫌だ/変だ」と判断できる観察可能な兆候。雰囲気・不穏だけ禁止
  - 「理由」系: 因果が1文で回収できる
  - 「瞬間」系: その瞬間の動作・反応を核に
  - 「口癖/一言」系: 発話そのものがズレていること
  - 「こんな○○は嫌だ」系: ○○の役割・作法・安心感のどれを壊すか先に決める
- **中心素材**を6つ書き出せ（場所、役職、備品、手続き、定番動作、定番の音/紙）。**主軸そのものは入れるな**
- 中心素材は**オチの核として使用禁止**（補助語に留める）。文の主語・目的語・変化対象・先頭にあれば差し替え。ただし主軸の素材（口癖の問いなら「発話」、理由の問いなら「因果接続」）は外すな

**1-d: 類縁語まで封印**（盛り塩→お札・数珠／湯呑み→急須・茶托）

**1-e: SKILL内の例示語を最終回答で使うな**（このSKILL内の具体名詞・数値・言い回しは方向付け用の足場）

### Step 2: 源泉ドメインを巡回して核を列挙

Claude系は **15核**、gpt系は **30核**。各核は**1コマで描ける具体物・動作・音・感覚**。

- **最初に浮かんだ前半を捨て、tail から取れ**（Claude: 前半7個捨て／gpt: 前半15個捨て）
- 各ドメインの「ヒント」は方向付けで、**その語をそのまま使うのは禁止**

| # | ドメイン | ヒント（この語自体は使うな） |
|---|---|---|
| 1 | 身体・生理音 | 末端の部位、本人が自覚しない反応・音 |
| 2 | 環境音・沈黙 | 設備の予定外の音、沈黙の理由 |
| 3 | 匂い・手触り・温度 | 料理以外の匂い、予想と違う質感、局所的な温度異常 |
| 4 | 物質・小道具 | 使い終わった後の小物、磨耗・補修の雑さ |
| 5 | 衣服・身辺 | 本人が気づいてない部分、本来あるべきものがない |
| 6 | 紙・書類・食 | 書かれ方が変な紙、扱いの荒さ／丁寧さの矛盾 |
| 7 | 視線・表情・声 | 一瞬の表情、視線の外し方、語尾・間・飲み込み |
| 8 | 数量・時間・儀礼の裂け目 | 半端な数、ズレ、遅れ、手順を一つ間違えた儀式 |

列挙後に Step 1 の封印リストと再照合し、引っかかったら別の核に置換。

### Step 2.5: 二次封印（手癖収束の検出）

核を並べて自問：

1. **共通手癖**：他のLLMが同じお題で挙げそうなものトップ5を具体名詞で挙げ、該当核を別に置換
2. **汎用怪談化**：身体反応の同期／物が体温を持つ／音が遅れる／発話の一文字異常／全員同方向／湿り気・ぬくもり・視線だけで不気味化 ── これらが混じったら破棄
3. **ベタ回避2周目の収束**：以下に着地していたら破棄
   - 儀式・神聖・祝詞・結界（異物に意味を与えて逃げる）
   - 倫理委員会・審議会・規定（制度語彙で誤魔化す）
   - 報告書・計画書・解析機・翻訳機・ご署名（手続きで止める）
   - 「ずっと○○し続け」「○○を繰り返す」（反復描写だけで不気味さを作る）
   - 「固まる」「フリーズ」「帰れない」（行動停止だけで成立させる）
   - 「○○人は△△するときに□□する」（観察報告の体で逃げる）
4. **数値手癖**：根拠なき精密数値（「3秒」「12回」等）を質的描写（「終わってからもまだ」「最後の一拍だけ」）に置換
5. **小欠陥チェック濫用**：穴・汚れ・濡れ・匂い・爪・靴下など「小さな異常」だけの核は、その場固有の制度・作法に巻き込まれていなければ落とせ
6. **ジャンルあるある**：「ヤバい○○」型で検索すれば10秒で出る典型あるある（上司なら「俺の若い頃は」「怒ってないよ」等）を回答の核に置くな

### Step 3: お題と接続

各候補をお題と接続できるか試せ。

- 接続の説明が**2文以上必要なら落とせ**
- お題の形容詞・動詞を別語で言い換えただけはベタ
- お題を別ジャンルに置き換えても成立する回答は汎用すぎ。落とせ

各候補について内部で1行作れ：

**「このお題の当たり前は何で、それをどう壊しているか」**

書けない候補は変な絵で大喜利ではない。一般的な不気味さ・不快さ・珍しさだけなら落とせ。

5回答のうち**最低3つ**は、その場固有の**制度・作法・客の期待**を壊すこと。
ただし中心素材そのものを変にせず、周辺の手順・反応・優先順位・扱われ方をズラせ。

### Step 3.5: 5本横並びチェック

5本を並べて一括確認。引っかかれば差し替え。

- **似せるな**：壊している対象／視点／文の圧 が同じなら片方差し替え
- **同じ嫌さに集めるな**：同じ「嫌さの作り方」が3本以上なら最弱を差し替え。集まりやすい型は **危険・脱出、監視・許可、変な説明、道具確認、遠い意味づけ**。危険・脱出系は5本中2本まで
- **嫌さの結果を散らす**：「この絵を見た人は何を失う・疑う・面倒に巻き込まれるから嫌か？」に答えろ。「なんとなく不穏」「変だから」だけなら落とせ。実務上の不利益（待つ・払う・選ぶ・移動する・扱われる・目的を邪魔される）まで届かせ、結果は **信頼・手間・居心地・金/時間・目的阻害** に散らす

### Step 4: 選抜（構造制約）

5つを選ぶ。次を全て満たせ。

- 5つは異なる源泉ドメインから（同ドメイン2つ不可）
- 同じ名詞・同じ構文が2回以上出ない
- 形式は最低4種類混ぜろ（体言止め／動詞止め／台詞単発／状況描写／擬音／問いかけ）
- ミニストーリー型（「Aと言いながらB」「直後にB」）は**最大2問**
- 時系列接続詞（「と言いながら」「と言った瞬間」「のあと」「直後に」）は**最大2回**

**接地の分散制約：**

物体（備品・建物・設備）接地は**最大2問**。残り3問以上は**人の側**から：

- 役割に反する動作（案内する人が避ける、作る人が疑う）
- 判断のズレ（優先順位、確認対象、止めるタイミングが変）
- 客・同僚への扱いのズレ（歓迎、説明、提供、引き継ぎが壊れる）
- 声・発話の歪み（語尾、間、イントネーション、飲み込み）
- 時間感覚の歪み（同期、遅れ、止まり）

「人の側」とは身体の小欠陥観察ではない。**その場で人が担うはずの役目が、どう間違って出ているか**。

**人側への寄せすぎ禁止：**

- 身体・衛生・生理的不快に接地は最大1問
- スタッフが失礼なだけは最大1問
- 客を検査・採点・監視は最大1問
- 同じ小道具・場所・職種・作法を2回使うな

### Step 5: 磨き

- 助詞を削れ（「〜が」「〜を」は大抵いらない）
- 主語が瞬時に確定する語順に
- 抽象名詞語尾（〜完備／〜対応／〜システム／〜問題）は具体物に戻せ
- 最後の2文字で決めろ

### Step 6: 最終自己批判

5つを並べて以下を問え。1つでも No なら Step 2 に戻れ。

1. 封印リスト（絶対禁止ワード + Step 1 + Step 2.5、類縁語含む）に戻っていないか？
2. 各回答が主軸（Step 1-c）に直接答えているか？
3. 回答だけ読んでお題が瞬時に逆算できるか？（出来ないなら接続が弱い＝滑り）
4. 各回答は1コマ漫画として描けるか？
5. 中心素材がオチの核になっていないか？（主軸の素材は核でよい）
6. 接地分散・形式分散・ミニストーリー上限を満たすか？
7. 嫌さの作り方・結果が散っているか？
8. 根拠なき精密数値が混入していないか？

### Step 7: 順番

- 回答1には**最も意外な角度**（最初に期待を裏切る）
- 回答3〜4に**最も自信のある**もの（中盤で仕留める）
- 自信のある候補を1番目に置くな

## NG

- お題と合っていない
- 差別・攻撃（皮肉・ブラックユーモア・自虐は武器）
- 説明しないと伝わらない
- 回答が長い
- 絵が浮かばない

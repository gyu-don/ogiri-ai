# ogiri-ai

Claude Code用の大喜利スキルです。お題に対して面白い回答を複数生成します。

## インストール

```bash
/plugin marketplace add https://github.com/gyu-don/ogiri-ai
/plugin install ogiri-ai
```

## 使い方

大喜利に関連するキーワードやお題を入力すると、自動的にこのスキルが起動します。

### 例

```
「こんなお年玉はいらない」というお題で大喜利して
```

```
この写真でボケて
```

または明示的にスキルを呼び出すこともできます：

```
/ogiri-ai AIに置き換えられない仕事
```
